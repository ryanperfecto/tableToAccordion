About Plugin

tableToAccordion enables a drop down effect to be added onto a traditional HTML table. It works by using the odd table row as a parent and the even row as a child by default. However, the order can be decided by adding and removing classes from the odd and even rows.

Usage

In order to use tableToAccordion you should create a table with a unique identifier (class or ID). Then call the plugin within a document.ready function, for example $('.TTA').tableToAccordion(); The following table provides a description of all of the options that can be passed to the plugin.

Option Description
parentClass This is the class that specifies the header of the table. The user will click on this area in order to show the child content.
